# Data structures in C

Data-structure implementations build from scratch in C

## Current

- Dynamic vector: allocation, append, automatic growth, bounds-check
  access, array display, and cleanup
- Tests: `tests/test_vector.c`

## Build and test

From this directory:

```sh
make vector
```

This builds and runs the vector test suite.

To remove generated build artifacts:

```sh
make clean
```
